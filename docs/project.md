# Основная идея

Создать приложение позволяющее вести переписки (чаты), как приватные (DM) так и групповые (GROUP_DM), также можно создавать звонки в этих чатах.
Также приложение позволяет создать гильдии (сервера), которые могут иметь свои чаты и группы для звонков.

# Авторизация

Основной вход происходит по username и password.
Планируется поддержка входа через сторонние сервисы, сначала только twitch.

Каждый пользователь имеет display_name - это отображаемое имя, а username - это уникальное имя по которому можно тегать пользователя.

# Чаты

## DM

desc: DIRECT_MESSAGES, прямые сообщения, личка
- Может существовать только 1 между каждой парой пользователей
- Создается автоматически при запросе, если на момент запроса такого не существует
- Считается, что канал не существует, пока в него не было отправлено хотя бы одно сообщение, если его удалить, то канал пропадет
- Каждый пользователь может создать личку сам с собой (сохраненные сообщения)
- При удалении, из канала удаляются все сообщения
- Каждый участник канала имеет право на полное удаление любых сообщений
- Оба участника имеют одинаковые разрешения, которые нельзя менять

## GROUP_DM

desc: Групповая переписка
- Имеет владельца, который имеет все права на группу
- Владельца можно передать, в таком случае, прошлый владелец становится администратором группы
- Можно назначать администраторов которым можно настроить права отдельно на каждого
- Приграсительные ссылки
  - Могут иметь ограниченное время
  - Можно задать имя ссылки
  - Опциональный лимит на число пользователей
- Введется аудит, его могут проссмотреть только администраторы и владелец

### Разрешения

Разрешения можно настроить на уровне группы, а также настроить их для отдельного пользователя.

Администраторам можно задать custom_title, которое будет отображаться заместо надписи "Admin"

- Отправлять сообщения (SEND_MESSAGES)
- Добавлять пользователей (ADD_MEMBERS)
- Закреплять сообщения (PIN_MESSAGES)
- Отправлять видео сообщения (SEND_VIDEO_MESSAGES)
- Отправлять голосовые сообщения (SEND_VOICE_MESSAGES)
- Отправлять файлы (SEND_FILES)
- Отправлять картинки (SEND_IMAGES)
- Создавать опросы (CREATE_POLLS)
- Обновлять информацию о канале (менять картинку, название, описание) (CHANGE_GROUP_INFO)

Только админские права:
- Удалять сообщения (DELETE_MESSAGES)
- Блокировать участников, разблокировать, просматривать список заблокированных (MANAGE_MEMBERS)
- Создавать/отзывать/просматривать ссылки-приглашения (MANAGE_INVITE_LINKS)
- Добавлять новых администраторов, с такими же правами или ниже (ADD_ADMINS)
